# Global-Retail-Sales-MIS-Dashboard

## Project Overview
This project is an Excel-based MIS dashboard created to analyze global retail sales performance.

## Objective
To prepare MIS reports and dashboards for sales analysis by category, sub-category, region, country, and payment mode.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Key Analysis
- Category and Sub-Category wise sales
- Payment Mode (Cash vs Online) analysis
- Country and Region wise sales
- Monthly sales trends

## Key Insights
- Electronics category contributes the highest sales
- Online payment mode has higher contribution
- Certain countries and regions perform better in sales

- ## Outcome
The dashboard helps management quickly understand sales performance and make data-driven decisions.
